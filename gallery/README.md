# Optitrek — Map Gallery

Curated interactive maps generated by the Optitrek pipeline. Each `.html` file is
a standalone Folium output that opens in any browser; the route data and markers
are inlined, only the basemap tiles (CartoDB Positron) require internet to load.

To view: double-click any `.html` file, or host the folder on GitHub Pages /
Vercel / Netlify for shareable links.

To regenerate: see `scripts/render_overlays.sh` and the individual experiment
scripts in `scripts/`. All require OSRM running on `localhost:5000` against
`data/osrm-major/` (see `CLAUDE.md` for setup).

---

## 01 — Tier 1: optimal NPS loop, capped (49 stops)

[`01_tier1_capped.html`](01_tier1_capped.html)

The main Tier 1 result. Solver constrained to exactly one stop per US zone
(48 contiguous states + D.C. = 49 zones, 49 stops). 466 NPS units in PostGIS,
filtered to 438 candidates (AK + HI excluded per the contiguous-US scope),
solved as a constrained set-cover + Hamiltonian cycle via Google OR-Tools.

**Result:** 49 stops · 49 states · **193.1 h · 9,744 mi** · 300s solver budget
· road geometry from self-hosted OSRM on the major-roads US extract.

vs Randal Olson's 2015 result (50 hand-picked stops, genetic algorithm, Google
Maps API → 224 h / 13,699 mi): **−13.8% driving time, −28.9% miles.**

## 02 — Tier 1: optimal NPS loop, uncapped (49 stops)

[`02_tier1_uncapped.html`](02_tier1_uncapped.html)

Same problem as 01, but the solver was allowed to add additional stops past
the one-per-zone minimum if doing so shortened the loop. It chose not to —
converged on the same 49-stop solution as the capped mode (within rounding).
Strong evidence that 49 stops is the geometric optimum on this candidate set.

**Result:** 49 stops · 49 states · **193.0 h · 9,756 mi** · 300s solver budget.

## 03 — Control 2: California double-stop

[`03_california_double_pseudo.html`](03_california_double_pseudo.html)

Apples-to-apples with Olson's trip shape: 50 total stops with 2 in California.
California POIs are split into CA-N (latitude ≥ 36.0°) and CA-S pseudo-zones,
forcing the solver to visit one from each. This isolates the "extra California
stop" factor from the broader Olson comparison.

**Result:** 50 stops · 49 zones (49 unique states + 1 extra CA) ·
**203.3 h · 10,367 mi** · 300s solver budget. The 2nd California stop costs
~10 h / ~600 mi vs the unconstrained 49-stop optimum.

## 05 — Four-way comparison

[`05_four_way_comparison.html`](05_four_way_comparison.html)

All four routes overlaid with toggleable layers:

- **Red:** Olson 2015 (50 hand-picked landmarks, GA, Google Maps)
- **Purple:** Control 1 — Optitrek's OR-Tools solver on Olson's exact 50 +
  his exact Google distances. Isolates pure optimizer quality. Result:
  221.7 h / 13,681 mi — only **−1.0% time / −0.1% miles** vs Olson.
- **Blue:** Optitrek capped (49 NPS, 193.1 h / 9,744 mi)
- **Green:** Optitrek California-double (50 NPS, 2 CA stops, 203.3 h / 10,367 mi)

What this decomposition shows: of the headline "−14% time, −29% miles" win,
≈1% comes from a more modern optimizer, ≈9% from algorithmic stop-selection
on a larger candidate pool (vs editorial curation of 50 landmarks), and ≈4%
from being free to drop the redundant 2nd California stop.

An 8-stop Western parks loop (Yellowstone → Grand Teton → Arches → Bryce

## 07 — Regional demo: New England NPS loop from Concord, NH

[`07_new_england_concord.html`](07_new_england_concord.html)

A regional variant: every NPS unit in the 6 New England states (CT, ME, MA,
NH, RI, VT), starting and ending at the New Hampshire State House in
Concord (the oldest US state house still in continuous legislative use,
107 N Main St, 43.207°N -71.538°W).

**Result:** 28 NPS units + 1 depot · 6 states · **44.9 h · 1,895 mi** ·
120s solver budget.

The solver picks a counter-clockwise sweep: VT (Marsh-Billings) → NH
(Saint-Gaudens) → MA interior (Springfield Armory, Coltsville) → CT (Weir
Farm) → RI cluster → coastal MA (New Bedford, Cape Cod) → Boston cluster
(14 contiguous stops including Boston NHP, Adams NHP, Salem Maritime, JFK,
Olmsted, Lowell) → ME north (Frances Perkins → Acadia → Roosevelt
Campobello → Saint Croix Island → Maine Acadian → Katahdin Woods) →
home. Demonstrates that the same pipeline handles arbitrary regional
sub-tours, not just the contiguous-US scope.

## 09 — Tier 2 demo: Southwest National Parks loop

[`09_southwest_parks.html`](09_southwest_parks.html)

The first Tier 2 demo: a 5-state National Parks loop generated from
`trips/southwest_parks.yaml`. Categories filtered to `national_park` only,
states limited to NM/AZ/UT/NV/CO. 14 candidate parks after filters;
solver selected the optimal 5-stop sub-tour. Demonstrates the config-driven
pipeline working end-to-end without any hardcoded scope.

**Result:** 5 stops · 5 states · **41.6 h · 1,431 mi** · 120s solver budget.

Generated via `python -m scripts.run_trip trips/southwest_parks.yaml`.
